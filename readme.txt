Send SMS texts to all your WordPress web-site's registered users in one click!

== Description ==

SubitoSMS WordPress Plugin lets you send SMS texts to your WP web-site's registered users, connecting to your SubitoSMS.it account.
It also modifies the basic WP user's registration form, adding a field for the user's mobile number.

= New features =
1. Allow to send SMS to all users when new post is created
2. Allow to select users to send SMS

== Installation ==

Installation is simple as a standard Wordpress Plugin.

e.g.

1. Download SubitoSMS-v1.x.zip file and upload to wordpress using Plugin menu 
2. Install from Wordpress Plugin's repository searching for SubitoSMS

== Frequently Asked Questions ==

= Are the SMS Free ? =

Unfortunately not, but we have very cheap prices.

= Do I need to sign-up ? =

Yes, registration is free and you can test with some free SMS messages.
